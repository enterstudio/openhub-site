Openhub

Openhub is a scalable and extensible distributed system that indexes open-source GitHub repositories written in Python, and analyzes their quality based on different quality attributes in order to later provide a more effective search of repositories based on their quality.

Learn more »

Quality Analysis

Currently, the web hosts millions of open source repositories. Finding the right repository amounts to searching repositories by name or description, but there is now way to search by quality attributes (e.g. performance, testability or usability). OpenHub aims to provide a searchable index of repositories and ther quality.
