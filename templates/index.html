Openhub

Openhub is a scalable and extensible distributed system that indexes open-source GitHub repositories written in Python, and analyzes their quality based on different quality attributes in order to later provide a more effective search of repositories based on their quality.

Learn more »

Quality Analysis

Currently, the web hosts millions of open source repositories. Finding the right repository amounts to searching repositories by name or description, but there is now way to search by quality attributes (e.g. performance, testability or usability). OpenHub aims to provide a searchable index of repositories and ther quality.

More info »

Repository Index

1.7 million GitHub projects were crawled up to now and more than 140,000 Python repositories analyzed for a total of 9.8TB of data processed using the OpenHub architecture. The resulting 3GB dataset collected is available for download, and is constanty growing.

Download the Data Set »

Extensible Code

OpenHub was built with extensibility in mind to allow develepers to write and integrate new quality analyzers for different quality attributes beyond the ones initially covered and for other languages beyond Python. We encourage people to fork the GitHub repository and submit pull requests, and new analyzers will be integrated into the architecture.
