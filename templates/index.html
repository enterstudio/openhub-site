Openhub

Openhub is a scalable and extensible distributed system that indexes open-source GitHub repositories written in Python, and analyzes their quality based on different quality attributes in order to later provide a more effective search of repositories based on their quality.

Learn more »

The Problem

Currently, the web hosts millions of freely accessible open source repositories. Finding the right repository amounts to searching repositories by name or description, and testing projects until a fit meets a desired set of quality attributes (e.g. performance, testability or usability). This is a time consuming task since there is no proper tool to search for repositories by how well they perform on a set of given quality attributes.
